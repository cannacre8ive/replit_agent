# CannaCre8ive Landing Page

Single-page, responsive marketing site optimized for cannabis operator lead generation.

## Files

- `index.html` — full landing page markup and styles.

## Run locally

Open `index.html` in a browser, or run:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy options

- GitHub Pages (static)
- Netlify (drag-and-drop or Git integration)
- Vercel static deploy
